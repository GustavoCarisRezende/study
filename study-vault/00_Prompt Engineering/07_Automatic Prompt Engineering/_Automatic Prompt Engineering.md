# Automatic Prompt Engineering

---
## **Resumo**
**Automatic Prompt Engineering (APE)** é uma abordagem que utiliza LLMs para **gerar** e **otimizar** prompts de forma automática, reduzindo o esforço manual na criação de prompts.
